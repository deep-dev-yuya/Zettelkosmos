# ⌨ ホットキー割当て一覧

## 🔧 Obsidian 本体機能
| 機能       | ホットキー     | 備考        |
| -------- | --------- | --------- |
| 新しいノート作成 | `Cmd + N` | 既定設定      |
| クイックスイッチ | `Cmd + O` | ファイル検索    |
| コマンドパレット | `Cmd + P` | コマンド検索／実行 |
| テスト      | `Cmd + N` |           |


## ⚡ QuickAdd
| 名前              | ホットキー         | 内容                         |
|-------------------|--------------------|------------------------------|
| プロパティ追加     | `Cmd + Ctrl + T`   | テンプレ「プロパティ」を実行 |

## ✨ Templater
| 名前                   | ホットキー         | 内容                              |
|------------------------|--------------------|-----------------------------------|
| 新規ノート作成（テンプレ） | `Cmd + Alt + T`     | Templaterのテンプレート新規作成用  |

---

## 📝 運用メモ

- ホットキーは増やしすぎない（3〜5個が理想）
- コマンドパレットから呼び出すものと住み分け
- 月1回程度、使用頻度を見直して整理
