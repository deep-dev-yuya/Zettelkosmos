> **備考:** 本プロジェクトでは、すべてのノートにプロパティ（YAML frontmatter）を毎回含める運用とする。

## プロジェクト概要

### 基本情報
- **プロジェクト名**: Zettelkosmos
- **目的**: Obsidianを使った知識管理システム
- **特徴**: 日本語でのノート管理、ガイド・マニュアル・テンプレートの体系化

### プロジェクト構造の理解
```

├── 06_🧠Notes/          # ノート・知見
├── 07_📓Logs/           # ログ・日記

```
